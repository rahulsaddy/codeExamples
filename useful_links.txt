Connecting to the Digital Ocean
Download the Public Key and Private Key files.
Install the Chrome SSH extension
In the "username@hostname or Free Form Text", type "vagrant@"
To import the SSH Key, click the "Import..." button and select the 'dataincubator' file you just downloaded. 
*Then, click again* and select the 'dataincubator.pub' file you just downloaded. For some odd reason, you need to select both. 
At this point you should see the identity dropdown value switch from '[default]' to 'dataincubator'
At this point you can click connect! If you're asked for a password, something has gone wrong, so please ask in the public channel on Slack.
The next step is the Ipython Notebook setup. Follow the instructions at the link.
